1. Redirect the response  *kulala.redirect-the-response-redirect-the-response*

You can redirect the response to a file.


DON’T OVERWRITE FILE*kulala.redirect-the-response-redirect-the-response-don’t-overwrite-file*

By using the `>>` operator followed by the file name, the response will be
saved to the file.

If the file already exists, a warning will be displayed, and the file won’t
be overwritten.

To overwrite the file, use the `>>!` operator.

title="do-not-overwrite.http"

>http
    POST https://httpbin.org/post HTTP/1.1
    Accept: application/json
    Content-Type: application/json
    
    {
      "name": "Kulala"
    }
    
    >> kulala.json
