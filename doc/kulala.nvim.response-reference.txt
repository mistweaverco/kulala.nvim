RESPONSE.HEADERS*kulala.nvim.response-reference-scripts-response-reference-response.headers*

Returns all response header objects.

Each header object has the following methods:

- `header.valueOf(headerName)` - Get the value of a header.
- `header.valuesOf(headerName)` - Retrieves the object containing all values of the headerName response header. Returns null if the headerName response header doesnâ€™t exist.

    client.log(response.headers.valueOf("Content-Type"));
